# Desaﬁo backend pleno
## Requisitos
- Java 8
- Maven
- Docker
- Docker Compose

## Windows
### Iniciar Execução
```bash
./execute.bat
```

### Parar execução
```bash
./stop.bat
```

### Logs
```bash
./logs.bat
```


### Unix-like
### Iniciar Execução
```bash
sh ./execute.sh
```

### Parar execução
```bash
sh ./stop.sh
```

### Logs
```bash
sh ./logs.sh 
```
